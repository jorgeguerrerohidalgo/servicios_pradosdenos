Configuración Inicial Requerida
Este proceso configura los tokens QR necesarios para que los guardias puedan hacer check-in en las plazas. Solo necesita ejecutarse una vez.
¿Qué hace este proceso?
  • Detecta plazas sin tokens QR
  • Genera tokens únicos para cada plaza
  • Los guarda en la base de datos
  • Permite generar códigos QR para imprimir
