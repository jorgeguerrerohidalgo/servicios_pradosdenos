Generar Tokens QR

Importante: Esta herramienta genera tokens QR únicos para todas las plazas que no tengan un token asignado. Los tokens son necesarios para que los guardias puedan hacer check-in escaneando los códigos QR.
Proceso:
  1. Se verifican las plazas activas sin token QR
  2. Se genera un token único para cada plaza
  3. Los tokens se guardan en la base de datos
  4. Se puede proceder a generar los códigos QR para impresión
