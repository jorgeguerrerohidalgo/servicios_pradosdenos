# 📚 Documentación del Proyecto

Esta carpeta contiene toda la documentación técnica del proyecto **Servicios Prados de Nos**.

## 📁 Estructura de Carpetas

### `/deployment`
- **Propósito**: Documentación relacionada con el despliegue en producción
- **Contenido**: Guías de deployment, configuraciones de producción, credenciales

### `/development`
- **Propósito**: Documentación para desarrolladores
- **Contenido**: Guías de desarrollo, configuraciones de entorno, mejoras de seguridad

### `/troubleshooting`
- **Propósito**: Guías de solución de problemas y diagnóstico
- **Contenido**: Fixes aplicados, instrucciones de debugging, soluciones conocidas

## 🔍 Cómo Usar Esta Documentación

1. **Para Deployment**: Revisar `/deployment` antes de cualquier despliegue
2. **Para Development**: Consultar `/development` para configurar el entorno
3. **Para Problemas**: Buscar en `/troubleshooting` antes de reportar bugs
